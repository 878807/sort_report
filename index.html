氣泡排序法 (Bubble Sort)

氣泡排序是一種簡單的排序演算法，其基本想法是重複地遍歷要排序的清單， 比較相鄰元素並在它們的順序錯誤時交換。這樣較大的元素會像氣泡一樣逐步"浮"到清單的頂端， 因而得名。

特性

  • 時間複雜度：O(n²)（最壞、平均情況）
  • 空間複雜度：O(1)，就地排序(in-place)
  • 當清單已經排序時，可使用旗標優化提前結束

Python 範例程式碼

def bubble_sort(arr):
    """
    Sorts an array using the bubble sort algorithm.
    Time complexity: O(n²)
    Space complexity: O(1)
    """
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
