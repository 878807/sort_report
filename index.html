氣泡排序視覺化

以下示範一個簡單的視覺化工具，讓你觀察氣泡排序如何透過比較與交換來整理數列。

Python 範例程式碼

def bubble_sort(arr):
    """
    Sorts an array using the bubble sort algorithm.
    Time complexity: O(n²)
    Space complexity: O(1)
    """
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
